### Prerequisites

-   **Node.js** (Latest LTS recommended)
-   **npm** or **pnpm/yarn**

### Setup

1. Clone the repository:

    ```sh
    git clone https://github.com/timjin/timjin.com
    cd timjin.com
    ```

2. Install dependencies:

    ```sh
    npm install
    ```

3. Start the development server:

    ```sh
    npm run dev
    ```

4. Build for production:
    ```sh
    npm run build
    ```
